# Отдельный Entry Point для ScheduleGenerator

## Описание

Отдельное приложение для генерации расписания кинотеатра, работающее независимо от основного приложения.

## Структура

### Файлы

- **main.tsx** - Entry point приложения
  - Импортирует реальные данные из `src/Content/Seance/schedule.ts` и `src/REDUX/filmsArray.ts`
  - Использует функцию `transformScheduleData()` для преобразования данных
  - Создает маппинги постеров и возрастных ограничений
  - Монтирует компонент `ScheduleImageGeneratorBatch` в DOM

- **gen.html** - HTML страница
  - Содержит `<div id="root"></div>` для монтирования React приложения
  - Подключает скрипт `main.tsx`

- **vite.config.js** - Конфигурация Vite
  - Настроена для двух entry points:
    - `index.html` → основное приложение
    - `gen.html` → приложение ScheduleGenerator

## Использование

### Разработка

```bash
npm run dev
```

Приложение будет доступно по адресу:
- Основное приложение: `http://localhost:8080/`
- ScheduleGenerator: `http://localhost:8080/gen.html`

### Сборка

```bash
yarn run build
```

Оба приложения будут собраны в папку `dist/`:
- `dist/index.html` - основное приложение
- `dist/gen.html` - приложение ScheduleGenerator

### Предпросмотр

```bash
yarn run preview
```

## Как работает

1. **Загрузка данных**
   - Расписание загружается из `src/Content/Seance/schedule.ts`
   - Данные фильмов загружаются из `src/REDUX/filmsArray.ts`
   - Функция `transformScheduleData()` преобразует данные в нужный формат

2. **Создание маппингов**
   - `filmMapping` - связывает названия фильмов с путями к постерам
   - `ageRatingMapping` - связывает названия фильмов с возрастными ограничениями

3. **Рендеринг**
   - Компонент `ScheduleImageGeneratorBatch` отображает интерфейс
   - Пользователь может выбрать день и скачать изображение расписания

## Обновление данных

Для обновления данных фильмов отредактируйте массив `filmsArray` в `src/REDUX/filmsArray.ts`:

```typescript
export const filmsArray = [
  {
    title: 'Название фильма',
    age: '6+',
    link: 'film_link',
    // ... другие поля
  },
  // ...
];
```

Для обновления расписания отредактируйте объект `schedule` в `src/Content/Seance/schedule.ts`:

```typescript
const schedule = {
  day0: [
    ['10:30', 'Название фильма 2D', '6+', '370₽'],
    // ...
  ],
  // ...
};
```

Убедитесь, что:
- `title` в filmsArray совпадает с названием в schedule (без суффикса "2D")
- `link` соответствует имени файла постера в `/Images/description/`
- `age` содержит возрастное ограничение
